Projeto Jovem Programador

O projeto é focado no ensino de programação de jogos para adolescentes do 9º Ano do Ensino Fundamental da escola E.E. Cônego Manoel Alves, no laboratório de informática do colégio.

Serão ministrados workshops semanais para uma turma, com duração de duas horas, totalizando seis aulas que serão realizadas com a ferramenta de programação Construct 3.

Time: André Luiz, Caroline Lucas, Mayara Fustaino e Pedro Artico
